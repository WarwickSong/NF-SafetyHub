APIKey 管理

阶段 6：支持手动录入和 KeyProvider 联通中转站创建；模型/token/资源权限由中转站管理。

新建 APIKey

手动录入已有中转站 Key，或一键由 KeyProvider 创建。默认 K-Sync：客户端 Key 与中转站 Key 一致。

Provider 创建会在创建成功后展示完整 Key；取消勾选后，SafetyHub 会自动生成独立客户端 Key。

APIKey 列表

点击「编辑」可修改名称、所属用户、部门、成本中心和过期时间；模型/token/资源权限由中转站管理。
