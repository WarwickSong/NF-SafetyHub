# 科普流程图

本目录是 SafetyHub 的"科普版"流程图，每张图节点少、用日常语言，适合：

- 给业务方 / 非研发同事 介绍系统
- 培训开场、领导汇报
- 新人 onboarding 的第一份阅读材料

如果想看与代码对齐、带技术细节的版本，请看姊妹目录 [机制流程图/](../机制流程图/README.md)。

## 推荐讲述顺序

**开场 4 张组合**：S1 → S2 → S5 → S6
- S1 立人设：是什么
- S2 切用户：我会怎么用
- S5 切环境：装在哪
- S6 切安全：会不会被乱拦

**深入 3 张组合**：S4 → S3 → S7
- S4 讲 Key 治理（为什么不会泄露）
- S3 讲管理员能力（出问题能不能追）
- S7 讲性能（会不会卡）
