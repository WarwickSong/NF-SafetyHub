# S2. 一次对话发生了什么（用户视角）

> 我点一下"发送"，背后到底走了什么流程。

```mermaid
flowchart TD
    Send(["我点击 发送"])
    Send --> Check["前哨站看一眼内容"]
    Check --> Judge{"内容安全吗？"}

    Judge -- "干净 ✅" --> Pass["原话转给大模型"]
    Judge -- "有点敏感 ⚠️" --> Mask["偷偷打码<br/>(例: 138****1234)<br/>再转给大模型"]
    Judge -- "明显违规 ❌" --> Block["拦下来<br/>不发给大模型<br/>直接假装大模型回答<br/>『抱歉，内容可能涉及敏感信息』"]

    Pass --> Reply1["大模型回答"]
    Mask --> Reply2["大模型回答<br/>(它看到的是打码后的版本)"]
    Block --> Fake["前哨站自己写的安全提示"]

    Reply1 --> Done(["我看到回答"])
    Reply2 --> Done
    Fake --> Done

    classDef ok fill:#dff0d8,stroke:#27ae60,color:#000
    classDef warn fill:#fff3cd,stroke:#b7791f,color:#000
    classDef bad fill:#fde2e2,stroke:#c0392b,color:#000
    class Pass,Reply1 ok
    class Mask,Reply2 warn
    class Block,Fake bad
```

## 三种结局对照

| 结局 | 我感受到的 | 实际发生的 |
|------|-----------|-----------|
| ✅ 通过 | 和直接用 ChatGPT 一样 | 内容原样转出去 |
| ⚠️ 脱敏 | 和平时一样收到回答 | 大模型其实只看到了打码版 |
| ❌ 拦截 | 收到一段"抱歉..."的回答 | 大模型其实**根本没看到这次提问** |
